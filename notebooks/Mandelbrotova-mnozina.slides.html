Mandelbrotova množina¶

Mandelbrotova mnoina je mnoina bod komplexn roviny, kter jsou odvozeny od rekurzivnch proces s komplexnmi sly patcmi tto mnoin a jejmu okol. Mandelbrotova mnoina je jeden z nejznmjch fraktl, pesnji eeno fraktlem je jej okraj. K jejmu uren se pouv zobrazen, kter kadmu komplexnmu slu $c$ piazuje uritou posloupnost komplexnch sel $z_n$. Tato posloupnost je urena nsledujcm rekurzivnm pedpisem:

$$z_0 = 0, \quad z_{n+1} = z_n^2 + c$$

Madelbrodova mnoina je pak definovna jako mnoina komplexnch sel $c$, pro kter je posloupnost $z_0, z_1, z_2, \dots$ omezen, tj. e spluje nsledujc podmnku:

Existuje reln slo $m$ takov, e pro vechna $n$ je $|z_n| \le m$.

Lze dokzat, e pekro-li absolutn hodnota nkterho lenu posloupnosti $z_n$ hodnotu 2, pak tato poslupnost nen omezen (jde do nekonena). Odtud je zejm, e lze ve ve uveden definici poloit $m = 2$, ani by tm dolo ke zmn jejho vznamu.

Jen pro zajmavost uvdme funkn jinak stejn kd, ale psan dve v minulosti velice populrnm jazykem FORTRAN.

program Mandelbrot
use types, only: dp
use constants, only: I
use utils, only: savetxt, linspace, meshgrid
implicit none

integer, parameter :: ITERATIONS = 100
integer, parameter :: DENSITY = 1000
real(dp) :: x_min, x_max, y_min, y_max
real(dp), dimension(DENSITY, DENSITY) :: x, y
complex(dp), dimension(DENSITY, DENSITY) :: c, z
integer, dimension(DENSITY, DENSITY) :: fractal
integer :: n
x_min = -2.68_dp
x_max = 1.32_dp
y_min = -1.5_dp
y_max = 1.5_dp

call meshgrid(linspace(x_min, x_max, DENSITY), &
    linspace(y_min, y_max, DENSITY), x, y)
c = x + I*y
z = c
fractal = 255

do n = 1, ITERATIONS
    print "('Iteration ', i0)", n
    where (abs(z) <= 10) z = z**2 + c
    where (fractal == 255 .and. abs(z) > 10) fractal = 254 * (n-1) / ITERATIONS
end do

print *, "Saving..."
call savetxt("fractal.dat", log(real(fractal, dp)))
call savetxt("coord.dat", reshape([x_min, x_max, y_min, y_max], [4, 1]))
end program
